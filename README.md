# 📊 Learning Matplotlib

This repository contains my hands-on practice and learning journey with Matplotlib, one of Python's most popular data visualization libraries.

## 🚀 Topics Covered

### 1. Basic Line Plot
- Creating first graph using Matplotlib
- Plotting numerical data

### 2. Multiple Line Plot
- Plotting multiple lines on the same graph
- Understanding legends and labels

### 3. Bar Charts
- Vertical bar charts
- Comparing categorical data

### 4. Horizontal Bar Charts
- Visualizing data horizontally
- Improving readability for categories

### 5. Scatter Plots
- Understanding relationships between variables
- Data point visualization

### 6. Pie Charts
- Showing proportions and percentages
- Visualizing category distributions

### 7. Histograms
- Understanding data distribution
- Working with bins and frequency ranges
